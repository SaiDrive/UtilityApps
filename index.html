Base64

This tool encodes/decodes base64 string. The term Base 64 is generic, and there are many implementations. MIME, which stands for Multi-Purpose Internet Mail Extensions, is the most common that is seen today. It is used to transmit attachments via email over the Simple Mail Transfer Protocol (SMTP). You should use Base64 whenever you intend to transmit binary data in a textual format

>

Hash Generator

This tool is one stop shop for various hash generations. It supports below hash generations utilities:

  1. MD5 Hash Generator : An MD5 hash is created by taking a string of an any length and encoding it into a 128-bit fingerprint.
  2. SHA-1 Hash Generator : SHA-1 (Secure Hash Algorithm) is a 160 bit cryptographic hash function created by the NSA in 1995
  3. SHA-256 Hash Generator : SHA256 is designed by NSA, it's more reliable than SHA1.
  4. SHA-512 Hash Generator : The SHA512 hash can not be decrypted if the text you entered is complicated enough.

RGB to Hex converter

It takes input in the form of values for Red, Green and Blue ranging from 0 to 255 and then converts those values to a hexadecimal string that can be used to specify color in html/css code. Photo editing software usually represents color in RGB and therefore if you would like to use the colors you use in your photo editing software as the background of your html element then you will have to get the hexadecimal representation of the RGB values. This tool allows you to get those values.
